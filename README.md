### Level 1: 基础与态射 (Foundations & Morphisms)
核心概念： 函数是一等公民、函数组合 (Composition)、恒等函数 (Identity)。

目标： 理解什么是范畴（对象是类型，态射是函数），以及如何组合它们。

练习： 实现一个通用的组合函数。

### Level 2: 函子 (Functor)
核心概念： 上下文 (Context)、映射 (Map/Transform)。

目标： 理解如何在一个“盒子”内部操作数据，而不打开盒子。

练习： 实现一个 Box<T> 类（或 Container<T>），并实现 map 方法。

### Level 3: 也许函子 (The Maybe Functor)
核心概念： 处理副作用（如空值）。

目标： 将 null 检查抽象化。

练习： 实现一个自定义的 Maybe<T> 类，处理存在或不存在的值，并支持 map。

### Level 4: 单子初步 (Monad - The "flatMap")
核心概念： 扁平化 (Flatten)、链式调用 (Chaining)。

目标： 解决嵌套盒子的问题 (Box<Box<T>>)，实现依赖计算。

练习： 给 Maybe<T> 添加 flatMap (或 bind) 方法。

### Level 5: 半群与幺半群 (Semigroup & Monoid)
核心概念： 结合律 (Associativity)、单位元 (Identity Element)。

目标： 理解如何把一堆对象“折叠”或“聚合”成一个。

练习： 实现通用的 fold 操作，处理整数加法、字符串拼接等。

### Level 6: 错误处理单子 (Either/Result Monad)
核心概念： 分支逻辑、纯函数式错误处理。

目标： 彻底告别 try-catch 和 throw。

练习： 实现 Result<T, E>，模拟成功的路径和失败的路径。
